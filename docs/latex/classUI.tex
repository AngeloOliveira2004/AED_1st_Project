Processes user requests for changes based on a given list of requests. 

Process a vector of requests. This method processes a vector of requests, where each request is a combination of a student, \mbox{\hyperlink{classUC}{UC}}, and an operation. The method identifies the operation and performs the corresponding action, such as adding a \mbox{\hyperlink{classUC}{UC}} to a student or removing a class from a \mbox{\hyperlink{classUC}{UC}}. After processing requests, it may set the Changes\+Made flag to indicate that changes were made.
