◆ process_requests()

void UI::process_requests ( std::vector< std::variant< Student, UC, char > >  requests_)

Processes user requests for changes based on a given list of requests.

Process a vector of requests. This method processes a vector of requests, where each request is a combination of a student, UC, and an operation. The method identifies the operation and performs the corresponding action, such as adding a UC to a student or removing a class from a UC. After processing requests, it may set the ChangesMade flag to indicate that changes were made.

Parameters
requests_A list of variant objects representing requests.
requests_A vector of requests to be processed.
